## 👨‍💻 About Me

I am a software developer from Brazil, building web applications since 2016. While my background covers the full stack, my true passion and daily expertise lie in the **modern Frontend ecosystem**, specifically working with React.js and Next.js.

I don't just write code; I focus on building scalable, maintainable, and highly performant applications. I prioritize high-quality engineering standards, actively applying **SOLID principles, Clean Code**, and clear architectural boundaries in my projects.

**My current technical focus:**
- Building robust interfaces with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.
- Modeling data and backend integrations using **Prisma** and **Drizzle ORM**.
- Maintaining an efficient, terminal-driven workflow on **Linux (Fedora)**, integrating modern AI agents to accelerate development and problem-solving.
