## 👨‍💻 Sobre Mim

Desenvolvedor de software com foco em **Frontend**, construindo aplicações web desde **2016**. Minha especialidade está no ecossistema moderno do React — de interfaces ricas até arquiteturas full-stack com Next.js.

Não me limito a "fazer funcionar": busco escrever código **limpo, testável e escalável**, aplicando ativamente princípios **SOLID, Clean Code** e separação de responsabilidades. Trabalho com Linux (Fedora) como ambiente principal e integro agentes de IA no fluxo de desenvolvimento para acelerar entregas sem abrir mão da qualidade.

> **Disponível para:** oportunidades remotas de Frontend, projetos freelance desafiadores e colaborações open-source.
